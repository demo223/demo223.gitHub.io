# Java中操作Redis

# 说明

前面我们讲解了Redis的常用命令，这些命令是我们操作Redis的基础，那么我们在java程序中应该如何操作Redis呢？这就需要使用Redis的Java客户端，就如同我们使用JDBC操作MySQL数据库一样。

Redis 的 Java 客户端很多，官方推荐的有三种：

  • Jedis
  • Lettuce
  • Redisson

Spring 对 Redis 客户端进行了整合，提供了 Spring Data Redis，在Spring Boot项目中还提供了对应的Starter，即 spring-boot-starter-data-redis。

# Jedis

Jedis 是 Redis 官方首选的 Java客户端开发包，使用简洁、方便。

操作步骤： ① 引入Jedis 的 maven 坐标

<dependency>
	<groupId>redis.clients</groupId>
	<artifactId>jedis</artifactId>
	<version>2.8.0</version>
</dependency>

使用 Jedis 操作 Redis 的步骤：

  1. 获取连接

    创建一个Jedis对象，直接new对象即可，代表获取到一个redis客户端连接

  2. 执行操作

    调用相关方法执行操作：set、get、hset、sadd

  3. 关闭连接

    关闭连接对象，调用 close方法

# 使用Jedis操作Redis

@Test
public void RedisTest() {
   // 1.获取Redis连接
   Jedis jedis = new Jedis("localhost", 6379);
   // 2.执行操作
   // 2.1操作String类型数据
   // 添加数据
   jedis.set("username", "ZhangSan");
   jedis.set("password", "123456");
   // 查询数据
   String username = jedis.get("username");
   String password = jedis.get("password");
   System.out.println(username);  //  ZhangSan
   System.out.println(password);  //  123456
   // 删除数据
   jedis.del("username", "password");
   // 2.2操作hash类型数据
    //  添加数据
   jedis.hset("001","username","ZhangSan");
   jedis.hset("002","password","123456");
   // 查询数据
   String username1 = jedis.hget("001", "username");
   String password1 = jedis.hget("002", "password");
   System.out.println(username1);  //  ZhangSan
   System.out.println(password1);  //  123456
   
	// 3.关闭连接
	jedis.close();
}

# 使用Spring Data Redis 操作Redis

Spring Data Redis是Spring的一部分，其中提供了一个RedisTemplate类对底层API进行了高度封装，来简化 Redis操作，将同一类型操作封装为operation接口，如下：

  • ValueOperations：简单Key-Value操作
  • SetOperations：set类型数据操作
  • ZSetOperations：zset类型数据操作
  • HashOperations：针对hash类型的数据操作
  • ListOperations：针对list类型的数据操作

# 配置文件

第一步：创建maven项目，配置 pom.xml文件，导入如下坐标

<dependencies>
    <dependency>
        <groupId>org.springframework.boot</groupId>
        <artifactId>spring-boot-starter-data-redis</artifactId>
    </dependency>
</dependencies>

第三步：提供配置类

@Configuration
public class RedisConfig extends CachingConfigurerSupport {

    @Bean
    public RedisTemplate<Object, Object> redisTemplate(RedisConnectionFactory connectionFactory) {

        RedisTemplate<Object, Object> redisTemplate = new RedisTemplate<>();

        //默认的Key序列化器为：JdkSerializationRedisSerializer
        redisTemplate.setKeySerializer(new StringRedisSerializer());
        redisTemplate.setHashKeySerializer(new StringRedisSerializer());

        redisTemplate.setConnectionFactory(connectionFactory);

        return redisTemplate;
    }
}

JDK的默认序列化方式为org.springframework.data.redis.serializer.JdkSerializationRedisSerializer

从RedisTemplate的源代码可以看出

在未设置序列化模式的时候，默认的序列化模式为JdkSerializationRedisSerializer

绝大多数情况下，不推荐使用 JdkSerializationRedisSerializer 进行序列化。主要是不方便看数据。

例如，我们在未配置序列化模式的时候在插入key - value 后，根据 key查找指定的value 的时候

public void teststring(){
    //存值
    redisTemplate.opsForValue().set("name","ZhangSan");
}

# 代码练习

# 操作 String 数据类型的数据

@Test
public void testString() {
   //存值
   redisTemplate.opsForValue().set("name", "ZhangSan");

   //取值
   String name = (String) redisTemplate.opsForValue().get("name");
   System.out.println(name);   //  ZhangSan

   //存值，同时设置过期时间
   redisTemplate.opsForValue().set("age", "23", 60, TimeUnit.SECONDS);

   //存值，如果存在则不执行任何操作返回false,不存在则存入并返回true
   Boolean flag = redisTemplate.opsForValue().setIfAbsent("city", "nanjing");
   System.out.println(flag);   //  true
}

# 操作 Hash 类型数据

@Test
public void testHash() {
   HashOperations hashOperations = redisTemplate.opsForHash();

   //存值
   hashOperations.put("001", "name", "ZhangSan");
   hashOperations.put("001", "age", "24");
   hashOperations.put("001", "address", "WH");

   //取值
   String name = (String) hashOperations.get("001", "name");
   String age = (String) hashOperations.get("001", "age");
   System.out.println(name);   //  ZhangSan
   System.out.println(age);    //  24

   //获得hash结构中的所有字段
   Set keys = hashOperations.keys("001");
   keys.forEach(System.out::println);  //  name age address


   //获得hash结构中的所有值
   List list = hashOperations.values("001");
   list.forEach(System.out::println);  //  ZhangSan 24 WH
}

# 操作 List 类型数据

@Test
public void testList() {
   ListOperations listOperations = redisTemplate.opsForList();

   //存值
   listOperations.leftPush("list", "a");
   listOperations.leftPushAll("list", "b", "c", "d");

   //取值
   List<String> list = listOperations.range("list", 0, -1);
   list.forEach(System.out::println);  //  d c b a

   //获得列表长度
   Long size = listOperations.size("list");
   int intSize = size.intValue();
   for (int i = 0; i < intSize; i++) {
      // 弹出并返回右边的元素
      System.out.println(listOperations.rightPop("list"));
      //  a b c d
   }
}

# 操作 Set 类型的数据

@Test
public void testSet() {
   SetOperations setOperations = redisTemplate.opsForSet();

   //存值
   setOperations.add("set", "a");
   setOperations.add("set", "a", "b", "c");

   //取值
   Set set1 = setOperations.members("set");
   set1.forEach(System.out::println);  //  c a b

   setOperations.remove("set", "b", "c");

   //取值
   Set set2 = setOperations.members("set");
   set2.forEach(System.out::println);  //  a
}

# 操作 ZSet 类型的数据

@Test
public void testZset() {
   ZSetOperations zSetOperations = redisTemplate.opsForZSet();

   //存值
   zSetOperations.add("zset", "a", 5);
   zSetOperations.add("zset", "b", 7);
   zSetOperations.add("zset", "c", 15);
   zSetOperations.add("zset", "a", 24);

   //取值    (分数是从小到大进行排序)
   Set<String> zset1 = zSetOperations.range("zset", 0, -1);
   zset1.forEach(System.out::println); //  b c a

   //修改分数
   zSetOperations.incrementScore("zset", "b", 50);

   //取值
   Set<String> zset2 = (Set<String>) zSetOperations.range("zset", 0, -1);
   zset2.forEach(System.out::println); //  c a b

   //删除成员
   zSetOperations.remove("zset", "a", "b");

   //取值
   Set<String> zset3 = zSetOperations.range("zset", 0, -1);
   zset3.forEach(System.out::println); //  c
}

# 操作通用类型

@Test
public void testCommon() {
   //获取Redis中所有的key
   Set<String> keys = redisTemplate.keys("*");
   keys.forEach(System.out::println);  //  zset set list

   //判断某个key是否存在
   Boolean hasKey = redisTemplate.hasKey("001");
   System.out.println(hasKey); //  true

   //删除指定key
   redisTemplate.delete("001");

   //获取指定key对应的value的数据类型
   DataType dataType = redisTemplate.type("zset");
   System.out.println(dataType.name());    //  ZSET
}
