# IO流

# 字节流和字符流

在Java中，流是从源读取并写入目标的数据序列。

从数据传输方式或者说是运输方式角度看，可以将 IO 类分为:

  • 字节流
  • 字符流

字节是个计算机看的，字符才是给人看的

# File类

# 创建File对象

File对象代表磁盘中实际存在的文件和目录。

创建File实例的方式常见的有以下3种

  1. 通过将给定路径名字符串转换成抽象路径名来创建一个新 File 实例

    File f1 = new File("D:/桌面/java测试.txt");
    1

  2. 根据 parent 路径名字符串和 child 路径名字符串创建一个新 File 实例

    File f2 = new File(":D/桌面/java的IO流文件夹", "Test1.txt");
    1

    相当于D/桌面/java的IO流文件夹/Test1.txt

  3. 通过给定的父抽象路径名和子路径名字符串创建一个新的File实例

    File parentDir = new File("D:/桌面/java的IO流文件夹");
File f3 = new File(parentDir, "Test1.txt");
    1
    2

    相当于D/桌面/java的IO流文件夹/Test1.txt

# File对象的方法

File的常用方法主要分为获取功能、获取绝对路径和相对路径、判断功能、创建删除功能的方法

创建一个File的实例化对象

File file = new File("D:/桌面/java测试.txt");
1
操作 方法
返回由此抽象路径名表示的文件或目录的名称 public String getName(file)
返回此抽象路径名的父路径名，
如果此路径名没有指定父目录，则返回 null		 public String getParent(file)
将此抽象路径名转换为一个路径名字符串 public String getPath(file)
返回由此抽象路径名表示的文件的长度 public long length()
删除此抽象路径名表示的文件或目录(空文件夹) public boolean delete()
当且仅当不存在具有此抽象路径名指定的名称的文件时
创建由此抽象路径名指定的一个新的空文件。		 public boolean createNewFile()
创建此抽象路径名指定的一级文件夹 public boolean mkdir()
创建此抽象路径名指定的多级文件夹 public boolean mkdirs()
返回此抽象路径名所表示文件夹下面文件名所组成字符串数组 public String[] list()
返回一个File数组，包含此路径目录中的文件File对象。
注意: 只能返回当前目录下的一级文件(夹)		 public File[] listFiles()
返回此文件最后一次修改的时间 public long lastModified()
// 通过将给定路径名字符串转换成抽象路径名来创建一个新 File 实例
File f1 = new File("D:/桌面/java测试.txt");
// 1.返回由此抽象路径名表示的文件或目录的名称
System.out.println(f1.getName());   //  java测试.txt
// 2.返回此抽象路径名的父路径名的路径名字符串，如果此路径名没有指定父目录，则返回 null.
System.out.println(f1.getParent()); //  D:\桌面
// 3.将此抽象路径名转换为一个路径名字符串
System.out.println(f1.getPath());   //  D:\桌面\java测试.txt
// 4.返回抽象路径名的绝对路径名字符串
System.out.println(f1.getAbsolutePath());   //  D:\桌面\java测试.txt
// 5.测试此抽象路径名是否为绝对路径名
System.out.println(f1.isAbsolute());  //true
// 6.返回由此抽象路径名表示的文件的长度
System.out.println(f1.length());    //  66
