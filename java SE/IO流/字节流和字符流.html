# 字节流和字符流

# 字节流

字节流用于读取和写入单个字节（8位）的数据。

所有字节流类都派生自称为InputStream和OutputStream的基本抽象类。

# 字符流

字符流用于读取和写入单个数据字符。

所有字符流类都派生自基本抽象类Reader和Writer。

# 字节流和字符流的区别

  • 字节流读取单个字节，字符流读取单个字符
  • 字节流用来处理二进制文件(图片、音频、视频文件)，字符流用来处理文本文件(可以看做是特殊的二进制文件，使用了某种编码，人可以阅读)。
  • 字节流无缓冲区，直接作用于文件；字符流有缓冲区，作用于缓冲区

# FileWriter和FileReader

# 字符输出流

java.io.Writer抽象类是字符输出流的所有类的超类（父类），将指定的字符信息写出到目的地。它同样定义了字符输出流的基本共性功能方法。

FileWriter 类从 OutputStreamWriter 类继承而来。该类按字符向流中写入数据。可以通过以下几种构造方法创建需要的对象。

在给出 File 对象的情况下构造一个 FileWriter 对象。

FileWriter fw = new FileWriter(new File("文件路径"),append);
1

append表示是否追加信息，

  • 如果 append 参数为 true，则将内容写入文件末尾处，相当于追加信息。
  • 如果 append 参数为 false, 则覆盖文档重新写入。

write()方法API

方法 说明
void write(int c) 写一个字符
void write(char[] ch) 写入一个字符数组
void write(char[] ch, int off, int len) 写入字符数组的指定部分
void write(String str, int off, int len) 写一个字符串的一部分

flush()方法

作用：刷新流，将当前缓冲区的数据写到文件里面，可以再次添加数据。

close()方法

作用：默认会调用一次flush()刷新，相当于也会刷新一次流，不过在使用该方法后流就关闭了，不能再向文件中添加数据。

public class Test {
	public static void main(String[] args) throws IOException {
		FileWriter fw1 = new FileWriter(new File("src/测试文件.txt");
		fw1.write("好久不见\n" + "最近好吗\n" + "常联系\n");
		// 注意: 这里的数据还没有写到文件里，还只是在缓冲区
		fw1.flush(); //  刷新缓冲区
		fw1.write("再次添加\n");
		fw1.flush(); //  刷新缓冲区
		fw1.close(); //  释放资源

		FileWriter fw2 = new FileWriter(f1, true);
		fw2.write("向文件末尾写入");
		fw2.flush(); //	后面没有其他操作的情况下，可以省略flush操作，直接close即可
		fw2.close();
	}
}

# 字符输入流

FileReader类从InputStreamReader类继承而来。该类按字符读取流中数据。

可以通过以下几种构造方法创建需要的对象。

在给定从中读取数据的 File 的情况下创建一个新 FileReader。

FileReader(new File("文件路径"))
1

读取的方式有常见的2种

读取方式 方法
一次读取一个字符(返回字符的ASCII编码) FileReader.read()
一次读取一个字符数组的数据(返回当前读取的所有字符的个数) FileReader.read(char[] buffer)

当文件中已经没有可读的字符之后会返回-1

public class Test {
	public static void main(String[] args) throws IOException {
		// 字符流的读取
		FileReader fr = new FileReader(new File("D:/桌面/java的IO流文件夹/Test2.txt"));
        
        //	一次读取一个字符
		int a;	//	获取的单个字符的ascll编码值
		while ((a = fr.read()) != -1) {
			System.out.print((char) a);
		}
//	--------------------------------------------------------------------------------
        //	一次读取多个字符
        char[] ch = new char[1024];
		int length;

		while ((length = fr.read(ch)) != -1) {
			System.out.println(new String(ch,0,length));
		}
        
		// 关闭资源
		fr.close();
	}
}

字符流只能完成文本的操作，不能操作图片和视频等非文本文件

使用FileReader和FileWriter完成文本文件复制

public class Test3 {
	public static void main(String[] args) throws IOException {
		File f1 = new File("src/侠客行.txt");
		FileReader fr = new FileReader(f1);
		File f2 = new File("src/侠客行2.txt");
		FileWriter fw = new FileWriter(f2);
		int a;
		while ((a = fr.read()) != -1) {
			fw.write(a);
		}
		fw.flush();
		fw.close();
	}
}

# FileInputStream和FileOutputStream

# 字节输入流

java.io.InputStream抽象类是表示字节输入流的所有类的超类（父类），可以读取字节信息到内存中。它定义了字节输入流的基本共性功能方法。

FileInputStream继承了InputStream，是文件输入流，从文件中读取字节。

FileInputStream fis = new FileInputStream(new File("文件路径"));
1

应用举例：

    File file = new File("src/侠客行.txt");
    FileInputStream fis = new FileInputStream(file);
    int a;//存放读的字节，就是读的结果
    while ((a = fis.read()) != -1) {
        System.out.print((char) a);
    }
    fis.close();

注意：上面采用的一个一个读的读取方法，不同编码格式下，一个中文汉字的字节数是不一样的，以UTF-8格式的“应”字为例，其占字节数为3。所以在文件中的中文会出现乱码。

解决办法：采用BufferReader可以避免乱码的情况。

public class Test {
	public static void main(String[] args) throws IOException {
		FileInputStream fis = new FileInputStream("src/侠客行.txt");
		// 一次读取一个字节(汉字会出现乱码,因为汉字在utf-8的情况下占3个字节)
		int a;
		while ((a = fis.read()) != -1) {
			System.out.println((char) a);
		}
		// -------------------------------------------------
		byte[] bytes = new byte[1024];
		int length; //  当前读取的字节个数
		StringBuffer sb = new StringBuffer();
		while ((length = fis.read(bytes)) != -1) {
            sb.append(new String(bytes,0,length));
		}
		System.out.println(sb.toString());
		fis.close();
	}
}

# 字节输出流

java.io.OutputStream抽象类是表示字节输出流的所有类的超类（父类）

FileOutputStream是OutputStream的子类，用于将数据（以字节为单位）写入文件。

 FileOutputStream fos = new FileOutputStream(file,append);
1

append表示是否追加信息

  • 如果 append 参数为 true，则将内容写入文件末尾处，相当于追加信息。
  • 如果 append 参数为 false, 则覆盖文档重新写入。

数据写出write()方法

方法 说明
public void write(int a) 写一个字节出去
public void write(byte[] buffer) 写一个字节数组出去
public void write(byte[] buffer , int pos , int len) 写一个字节数组的一部分出去
public class Test2 {
    public static void main(String[] args) throws IOException {
        FileOutputStream fos = new FileOutputStream("src/写入数据.txt");
        // 写入一个字符编码值
        fos.write(97);  //  写入字符'a'
        // 写入一个字节数组
        byte[] bytes = new byte[]{97,98,99,100,101};
        fos.write(bytes);  //   写入字符'a','b','c',,d','e'
        // 写入字节数组的指定部分
        fos.write(bytes,1,3);   //  写入字符'b','c',,d'
        fos.close();    //  关闭流
        // 打印结果:  aabcdebcd
    }
}

使用FileInputStream和FileOutputStream完成文本文件复制

public class Test {
	public static void main(String[] args) throws IOException {
		FileInputStream fis = new FileInputStream("./src/侠客行.txt");
		FileOutputStream fos = new FileOutputStream("./src/侠客行2.txt");
		byte[] bytes = new byte[1024];
		int length;   //  读取的字节个数
		while ((length = fis.read(bytes)) != -1) {
			fos.write(bytes, 0, length);
		}
		fis.close();
		fos.close();
	}
}

public class Test8 {
	public static void main(String[] args) {
		try (FileInputStream fis = new FileInputStream("./src/图片.jpg"); 
             FileOutputStream fos = new FileOutputStream("./src/图片2.jpg");) {
			byte[] bytes = new byte[1024];
			int length;   //  读取的字节个数
			while ((length = fis.read(bytes)) != -1) {
				fos.write(bytes, 0, length);
			}sh
		} catch (IOException e) {
			throw new RuntimeException(e);
		}
	}
}
