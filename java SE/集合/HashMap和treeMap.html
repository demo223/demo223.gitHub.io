# HashMap和treeMap

Map 是一种键-值对（key-value）集合，Map 集合中的每一个元素都包含一个键（key）对象和一个值（value）对象。用于保存具有映射关系的数据。

Map集合里保存着两组值，一组值用于保存Map 里的key，另外一组值用于保存 Map里的 value，key和 value 都可以是任何引用类型的数据。Map 的 key 不允许重复，value 可以重复，即同一个 Map 对象的任何两个 key 通过 equals方法比较总是返回 false。

Map 中的 key 和 value 之间存在单向一对一关系，即通过指定的 key，总能找到唯一的、确定的 value。从 Map 中取出数据时，只要给出指定的 key，就可以取出对应的 value。

Map 接口主要有两个实现类：HashMap 类和 TreeMap 类。其中，HashMap 类按哈希算法来存取键对象，而 TreeMap 类可以对键对象进行排序。

在修改Map集合中的数据的时候，如果使用HashMap的put方法会覆盖之前的值，元素的位置也会发生改变，如果想维持之前的集合顺序，可以使用LinkedHashSet。

Map 集合最典型的用法就是成对地添加、删除key-value 对，接下来即可判断该Map中是否包含指定 key，也可以通过 Map 提供的 keySet()方法获取所有 key 组成的集合，进而遍历 Map 中所有的 key-value 对。.

下面程序示范了 Map 的基本功能。

public class MapStudy {
	public static void main(String[] args) {
		HashMap<String， String> hashMap = new HashMap<>();
		hashMap.put("姓名"， "张三");
		hashMap.put("语言"， "中文");
		hashMap.put("国籍"， "中国");
		hashMap.put("区域"， "湖北");
		hashMap.put("性别"， "男");
		hashMap.put("性别"， "女"); //  添加键值对时如果已经存在该key，则覆盖之前的键值对
		System.out.println(hashMap);
		// {姓名=张三， 语言=中文， 国籍=中国， 区域=湖北， 性别=女}
		System.out.println(hashMap.get("姓名"));  //  张三
		System.out.println(hashMap.remove("区域"));   //  湖北
		System.out.println(hashMap.remove("性别"， "女"));  //  true
		System.out.println(hashMap);    //  {姓名=张三， 语言=中文， 国籍=中国}
		Map<String， String> map = new HashMap<>();
		map.put("key1"， "value1");
		map.put("key2"， "value2");
		map.put("key3"， "value3");
		System.out.println(map);    //  key1=value1， key2=value2， key3=value3}
		hashMap.putAll(map);
		System.out.println(hashMap);
		// {key1=value1， 姓名=张三， key2=value2， 语言=中文， key3=value3， 国籍=中国}
		System.out.println(hashMap.containsKey("姓名"));  //  true
		System.out.println(hashMap.containsValue("中国"));    //  true
		// 返回 Map 集合中所有键对象的 Set 集合
		System.out.println(hashMap.keySet());   //  [key1， 姓名， key2， 语言， key3， 国籍]
		Set<String> s = hashMap.keySet();
		// 返回 Map 集合中所有值对象的 Collection 集合
		System.out.println(hashMap.values());
		Collection<String> c = hashMap.values();
		// 返回 Map 集合中所有键值对对象的 Set 集合
		System.out.println(hashMap.entrySet());
		Set<Map.Entry<String， String>> kv = hashMap.entrySet();
		// [key1=value1， 姓名=张三， key2=value2， 语言=中文， key3=value3， 国籍=中国]
		System.out.println(Map.entry("姓名"， "张三"));  //  姓名=张三
		// 删除Map集合中所有的键值对
		hashMap.clear();
		System.out.println(hashMap);    //  {}
		System.out.println(hashMap.isEmpty());  //  true
		System.out.println(hashMap.size()); //  0
	}
}

# TreeMap

# 使用TreeMap排序

根据键名进行排序

注意: TreeMap并不是根据插入的顺序来排序，而是根据key元素的 compareTo 方法来排序。

TreeMap集合默认会对键key进行排序

TreeMap<String， String> treeMap = new TreeMap<>();
// 键值对  <学号，姓名>
treeMap.put("006"， "张三");
treeMap.put("021"， "李四");
treeMap.put("003"， "刘备");
treeMap.put("036"， "关羽");
treeMap.put("014"， "张飞");
System.out.println(treeMap);
// {003=刘备， 006=张三， 014=张飞， 021=李四， 036=关羽}
// TreeMap集合默认会对键进行排序

# 自定义排序

TreeMap集合自定义排序规则有2种方式

  1. 在类中实现Comparable接口，重写compareTo方法。

  2. 在创建集合的实例对象的时候自定义一个比较器

    TreeMap<String, Integer> map = new TreeMap<>((a, b) -> a.compareTo(b));

遍历键值对

使用Map.keySet()方法来获取Map集合中所有的key(键值)，再根据key值通过get(key)方法来获取value。

HashMap<String，String> map = new HashMap<>();
map.put("张三"，"武汉");
map.put("李四"，"北京");
map.put("王五"，"上海");
Set<String> set = map.keySet();
for (String s : set) {
    System.out.println(map.get(s)); //  北京  武汉  上海
}
