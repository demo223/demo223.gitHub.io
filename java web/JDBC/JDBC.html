# JDBC

# JDBC概念

全称：Java DataBase Connectivity，Java数据库连接，就是使用Java语言操作关系型数据库的一套API规范

本质：JDBC 就是使用Java语言操作关系型数据库的一套API接口

# JDBC入门

编写步骤

  1. 注册驱动，注册程序具体的数据库驱动
  2. 获取连接 Connection
  3. 获取执行SQL的对象 PreparedStatement
  4. 执行SQL语句
  5. 处理结果
  6. 释放资源

JDBC的常用API

API 作用
DriverManager 用来注册驱动和获取数据库连接
Connection 获取执行SQL语句的对象和事务管理(提交(方式)/回滚)
PreparedStatement 执行SQL语句和设置SQl语句中的占位符参数(?)
ResultSet 封装了查询语句的结果

# 获取数据库连接

DriverManager.getConnection(String url, String user, String password)
1

例如下面获取本地数据库的连接

Connection conn = DriverManager.getConnection("jdbc:mysql://localhost:3306/db2?useSSL=false", "root", "123456");
1

Connection接口中定义了3个对应的方法

开启事务：setAutoCommit(true或false)：true为自动提交事务；false为手动提交事务

提交事务：commit()

回滚事务：rollback()

# 获取执行SQL语句的对象

PreparedStatement ps = 数据库连接.prepareStatement(SQL代码);
1

例如

PreparedStatement ps = conn.prepareStatement("update user set money = 5000 where name = '张三'");
1

PreparedStatement设置SQL中的占位符setXXX(int index, Object value)

解释：index表示设置的第几个参数(从1开始)，value表示传入的参数的值(与SQL中的?对应)

  • 执行修改语句(增删改语句)

# JDBC代码练习

模拟张三向李四转账1000块钱，如果转账发生异常就回滚。

id name money
1 张三 5000
2 李四 5000
public class Test {
	public static void main(String[] args) throws SQLException, ClassNotFoundException {
		//1 注册驱动，告诉程序使用哪个jar包
		Class.forName("com.mysql.jdbc.Driver");
		//2 获取连接
		Connection conn = DriverManager.getConnection("jdbc:mysql:///db2?useSSL=false", "root", "990610");
		PreparedStatement ps = null;

		try {
			// 开启事务
			conn.setAutoCommit(false);//开启手动提交

			//TODO 需求：事务管理：  张三向李四借500元
			String sql = "update user set money = money - ? where id = ?";
			ps = conn.prepareStatement(sql);

			// 张三借钱
			ps.setDouble(1, 1000);
			ps.setDouble(2, 1);
			int count1 = ps.executeUpdate();
			System.out.println(count1 > 0 ? "张三支出成功" : "张三支出失败");
			// 李四得钱
			ps.setDouble(1, -1000);
			ps.setDouble(2, 2);
			int count2 = ps.executeUpdate();
			System.out.println(count2 > 0 ? "李四收入成功" : "李四收入失败");

			// 查看数据库中张三和李四的余额
			String sql2 = "select money from user where name = ?";
			ps = conn.prepareStatement(sql2);
			ps.setString(1, "张三");
			ResultSet res1 = ps.executeQuery();
			while (res1.next()) {
				System.out.println("张三的当前余额为" + res1.getDouble("money"));
			}
			ps.setString(1, "李四");
			ResultSet res2 = ps.executeQuery();
			while (res2.next()) {
				System.out.println("李四的当前余额为" + res2.getDouble("money"));
			}

			// 提交事务
			conn.commit();

		} catch (Exception ex) {   //出现任何异常都要回滚
			ex.printStackTrace();
			// 回滚事务，程序在出现异常时会执行到这个地方，此时就需要回滚事务
			conn.rollback();
		}
		//5 释放资源
		if (ps != null) {
			ps.close();
		}
		conn.close();
	}
}

# 查询数据

public List<Brand> selectAll() throws Exception {
		//1 注册驱动，获取连接对象 Connection
		Connection conn = DriverManager.getConnection(url, username, password);
		//2 定义SQL
		String sql = "select * from brand";
		//3 获取PreparedStatement对象，预编译SQL
		PreparedStatement ps = conn.prepareStatement(sql);
		//4 如果SQL用有？占位符，就设置参数
		//5 执行SQL，获取结果
		ResultSet res = ps.executeQuery();
		//6 处理结果
		List<Brand> brandList = new ArrayList<Brand>();
		//6.1 获取每一行数据
		while (res.next()) {
			int id = res.getInt("id");
			String brandName = res.getString("brand_name");
			String companyName = res.getString("company_name");
			int ordered = res.getInt("ordered");
			String description = res.getString("description");
			int status = res.getInt("status");
			//6.2 封装成Brand对象
			//6.3 保存到List集合中
			brandList.add(new Brand(id, brandName, companyName, ordered, description, status));
		}
		//7 释放资源
		res.close();
		ps.close();
		conn.close();
		//返回结果
		return brandList;
	}

# 添加数据

public boolean add(Brand brand) throws Exception {
   //1 注册驱动，获取连接对象 Connection
   Connection conn = DriverManager.getConnection(url, username, password);
   //2 定义SQL
   String sql = "insert into brand  values(?,?,?,?,?,?)";
   //3 获取PreparedStatement对象，预编译SQL
   PreparedStatement ps = conn.prepareStatement(sql);
   //4 如果SQL用有？占位符，就设置参数
   ps.setInt(1, brand.getId());
   ps.setString(2, brand.getBrandName());
   ps.setString(3, brand.getCompanyName());
   ps.setInt(4, brand.getOrdered());
   ps.setString(5, brand.getDescription());
   ps.setInt(6, brand.getStatus());
   //5 执行SQL，获取结果
   int count = ps.executeUpdate(); //  受影响的条数
   //6 处理结果
   //7 释放资源
   ps.close();
   conn.close();
   //返回结果
   return count == 1;
}

public boolean update(Brand brand) throws Exception {

   return count == 1;
}

# 数据库连接池

  • 数据库连接池是个容器，负责分配、管理数据库连接(Connection)

  • 它允许应用程序重复使用一个现有的数据库连接，而不是再重新建立一个；

  • 资源重用

  • 提升系统响应速度

  • 之前我们代码中使用连接是没有使用都创建一个Connection对象，使用完毕就会将其销毁。这样重复创建销毁的过程是特别耗费计算机的性能的及消耗时间的。

  • 而数据库使用了数据库连接池后，就能达到Connection对象的复用，如下图

# DataSource

标准接口：DataSource 接口

官方(SUN) 提供的数据库连接池标准接口，由第三方组织实现此接口。

功能：获取连接

常见的数据库连接池：DBCP、C3P0、Druid

Druid(德鲁伊)

  • Druid连接池是阿里巴巴开源的数据库连接池项目
  • 功能强大，性能优秀，是Java语言最好的数据库连接池之一

    import com.alibaba.druid.pool.DruidDataSourceFactory;

import javax.sql.DataSource;
import java.io.InputStream;
import java.sql.Connection;
import java.sql.PreparedStatement;
import java.util.Properties;

/**
 * Druid数据库连接池演示
 */
public class DruidDemo {
    public static void main(String[] args) throws Exception {
        //1.导入jar包
        //2.定义配置文件
        //3. 加载配置文件
        Properties properties = new Properties();
        //读取文件，得到inputStream输入流
        InputStream is = DruidDemo.class.getClassLoader().getResourceAsStream("druid.properties");
        //加载数据
        properties.load(is);
        //4. 获取连接池对象
        DataSource dataSource = DruidDataSourceFactory.createDataSource(properties);
        //5. 获取数据库连接 Connection
        Connection conn = dataSource.getConnection();
        //6. 获取执行SQL语句的执行者对象
        PreparedStatement ps = conn.prepareStatement("update user set money=1000 where id=1");
        //7. 执行SQL语句
        int count = ps.executeUpdate();
        //8. 处理结果
        System.out.println("影响的行数 " + count);
        //9. 归还连接到连接池
        ps.close();
        conn.close();
    }
}
