# xxl-job-admin安装

  1. 拉取xxl-job-admin指定版本

    docker pull xuxueli/xxl-job-admin:2.3.1
    1

  2. 在centos上创建目录

    创建/data/soft/xxl-job

    创建/data/soft/xxl-job/applogs

    解压xxl-job-2.3.1.zip,找到 配置文件 application.properties

    将配置文件 拷贝到 /data/soft/xxl-job/application.properties

    修改数据 库连接 配置如下：

    spring.datasource.url=jdbc:mysql://192.168.101.65:3306/xxl_job2.3.1?useUnicode=true&characterEncoding=UTF-8&autoReconnect=true&serverTimezone=Asia/Shanghai
spring.datasource.username=root
spring.datasource.password=mysql
spring.datasource.driver-class-name=com.mysql.cj.jdbc.Driver
    1
    2
    3
    4

    注意：mysql的地址如果是宿主机的，不要写成localhost或者127.0.0.1了，要写真实IP地址。

  3. 创建容器

    docker run -d -p 8088:8080 --name xxl-job-admin -v /data/soft/xxl-job/application.properties:/application.properties -v /data/soft/xxl-job/applogs:/data/applogs  -e PARAMS='--spring.config.loation=/application.properties' xuxueli/xxl-job-admin:2.3.1
    1

# MySQL8安装

  1. docker拉取指定mysql版本

    docker pull mysql:8.0.26
    1

  2. 创建容器

    docker run -p 3306:3306 --name mysql \
-v /usr/local/docker/mysql/mysql-files:/var/lib/mysql-files \
-v /usr/local/docker/mysql/conf:/etc/mysql \
-v /usr/local/docker/mysql/logs:/var/log/mysql \
-v /usr/local/docker/mysql/data:/var/lib/mysql \
-e MYSQL_ROOT_PASSWORD=123 \
-d mysql:8.0.26
    1
    2
    3
    4
    5
    6
    7

  3. 设置mysql开启自启

    docker update mysql --restart=always
    1

  4. 进入mysql容器

    docker exec -it mysql bash
    1

  5. 退出容器

    exit
    1

  6. 登陆(默认密码root)，一定要先进入mysql容器

    mysql -uroot -proot
    1

  7. 重启mysql服务

    docker restart mysql
    1

  8. 启动mysql

    docker start mysql
    1

  9. 查看mysql的安装地方

    whereis mysql
    1

  10. 重启docker

    sudo systemctl restart docker
    1

docker运行期间如果重启防火墙，docker就会报iptables error，这个时候需要重启docker

终止MySQL进程： killall mysqld

云服务器必须先开放端口9092和2181     ！！！

# 2 zookeeper安装

Zookeeper是安装Kafka集群的必要组件，Kafka通过Zookeeper来实施对元数据信息的管理，包括集 群、主题、分区等内容。 同样在官网下载安装包到指定目录解压缩

ZooKeeper 官网： http://zookeeper.apache.org/

在提供的资料中，有一个zookeeper-3.4.14.tar.gz包，上传到服务器，也可以到官网上下载

（1）解压压缩包

tar zxvf zookeeper-3.4.14.tar.gz
1

（2）修改配置文件，进入安装路径conf目录，并将zoo_sample.cfg文件修改为zoo.cfg

cd zookeeper-3.4.14  #进入安装目录
cd conf   #进入配置目录
mv zoo_sample.cfg zoo.cfg  # 把文件改名
1
2
3

（3）创建存放数据的目录 data

在zookeeper安装的根目录创建目录 data

mkdir data

进入conf目录下，编辑zoo.cfg

进入bin目录

# 3 kafka安装

（1）官网下载

下载地址：http://kafka.apache.org/downloads

也可以在今天的资源文件夹中找到这个安装，直接上传到服务器即可

（2）解压

tar zxvf kafka_2.12-2.2.1.tgz
1

（3）修改参数

修改config目录下的server.properties文件,效果如下

  • 修改listeners=PLAINTEXT://:9092
  • advertised.listeners=PLAINTEXT://自己的云服务器ip:9092
  • log.dirs=/root/kafka_2.12-2.2.1/logs 需要在kafka安装目录新建logs目录

（4）启动kafka

在kafka的根目录

bin/kafka-server-start.sh config/server.properties  #启动kafka
1

查看进程

注意：启动kafka之前，必须先启动zookeeper
