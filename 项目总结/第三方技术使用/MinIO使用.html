# MinIO使用

# 一、MinIO简介

MinIO基于Apache License v2.0开源协议的==对象存储服务==，可以做为云存储的解决方案用来保存海量的图片，视频，文档。由于采用Golang实现，服务端可以工作在Windows,Linux, OS X和FreeBSD上。配置简单，基本是复制可执行程序，单行命令可以运行起来。

MinIO兼容亚马逊S3云存储服务接口，非常适合于存储大容量非结构化的数据，例如图片、视频、日志文件、备份数据和容器/虚拟机镜像等，而一个对象文件可以是任意大小，从几kb到最大5T不等。

基本概念

  • bucket– 类比于文件系统的目录
  • Object– 类比文件系统的文件
  • Keys– 类比文件名

官网文档：http://docs.minio.org.cn/docs/

# 二、MinIO安装

  1. 拉取镜像

    docker pull minio/minio
    1

  2. 创建容器

    docker run -i -d \
  -p 9000:9000 \
  -p 9001:9001 \
  --restart=always \
  --name minio1 \
  --network=host \
  --privileged=true \
  -v /etc/localtime:/etc/localtime \
  -v ~/minio/data:/data \
  -e "MINIO_ROOT_USER=minio" \
  -e "MINIO_ROOT_PASSWORD=minio123" \
  quay.io/minio/minio server /data --console-address ":9001"

  4. 编写一个文件上传方法

    /***
 * 文件上传
 * @param is
 * @throws Exception
 */
public void upload(InputStream is) throws Exception {
   //  创建一个MinIO的客户端，并且配置账号信息、指定服务地址
   MinioClient client = MinioClient.builder()
                                   .credentials("minioadmin", "minioadmin")
                                   .endpoint("http://localhost:9000").build();
                                   // .credentials("minio", "minio123")
                                   // .endpoint("http://192.168.33.31:9000").build();

   //  封装文件信息，文件内容、文件上传到哪个Bucket，文件的名字，文件类型
   PutObjectArgs putObjectArgs = PutObjectArgs.builder()
                                              .stream(is, is.available(), -1).bucket("demo")
                                              .object("/2023/02/05/test.html").contentType("text/html")
                                              .build();

   //  使用MinIO的客户端将封装的文件数据上传到MinIO中
   client.putObject(putObjectArgs);
}

  5. 添加MinioController代码

    @Controller
@RequestMapping("/freemarker")
public class MinioController {
	@Autowired
	private Configuration configuration;

	@ResponseBody
	@GetMapping("/upload")
	public String html() throws Exception {
		//1：准备数据模型
		Map<String, Object> map = new HashMap<>();
		//构建数据模型的数据
		User user = new User("张三", 25);
		map.put("user", user);

		//2：准备模板,并使用Configuration读取模板
		Template template = configuration.getTemplate("1.ftl");

		//3：使用模板对象载入数据模型，并生成文件
		//template.process(map,new FileWriter("D:/桌面/123.html"));

		//获取生成的文件内容
		String content = FreeMarkerTemplateUtils.processTemplateIntoString(template, map);
		ByteArrayInputStream bais = new ByteArrayInputStream(content.getBytes());

		// 上传文件
		upload(bais);

		return "OK";
	}
}
